# atividade8
 
## Atividade de desafio final PHP

Teve como objetivo a criação de um formulário para cadastro para a compra de um ingresso. Foi utilizado HTML para a estrutura do formulário, com espaço para cadastro do nome, idade da pessoa e o tipo de ingresso. Foi utilizado o PHP para a criação de condições, como proibir a compra para menores de 18 anos e para que apareça o nome do comprador, junto com o ingresso que seria comprado e o valor do ingresso. Por fim, foi utilizado o CSS apenas para inserir uma imagem de fundo na página criada.
